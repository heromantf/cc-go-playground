## Build and Run Commands

```bash
# Run the application
go run main.go

# Build the application
go build -o cc-go-playground

# Run tests
go test ./...

# Run a single test
go test -run TestName ./...
```

## Project Overview

This is a Go playground/sandbox project for experimentation. It uses Go 1.25 and currently contains a simple main.go entry point.
